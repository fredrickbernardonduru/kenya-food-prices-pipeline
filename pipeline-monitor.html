How to use this monitor
This tool polls your services via their HTTP health endpoints. It never writes to your database, modifies DAGs, or restarts containers — read-only, zero interference.
01 / AIRFLOW
Open Airflow UI
Checks localhost:8081/health. Click the card to open the Airflow web interface in a new tab. Default login: admin / admin
02 / METABASE
Open Metabase
Checks localhost:3000/api/health. Click the Metabase card to go straight to your dashboards with maize price trends.
03 / POSTGRES
Database connection
Verifies TCP reachability on localhost:5432. If the card is red, run docker compose up postgres from the docker/ folder.
04 / REFRESH
Manual & auto-refresh
Checks run every 60s automatically. Hit the ↻ refresh button in the top bar to trigger an instant check at any time.
QUICK START — place this file and run:
# 1. Drop pipeline-monitor.html into your project root
# 2. Start your stack
cd docker && docker compose up -d
# 3. Open the monitor (no server needed)
open pipeline-monitor.html
